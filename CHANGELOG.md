# 更新日志

本项目所有重要变更均记录于此文件。版本号遵循语义化版本（SemVer）：主版本.次版本.修订号。

## [0.3.0] - 2026-06-01

V2 升级 · 阶段一（SaaS 真实对接 + 报表导出 + 国标税务申报表）。

### 新增
- **SaaS 系统真实对接**：新增统一对接层 `lib/saas-integration.ts`，支持储值、消耗、营业、薪酬、采购 5 类数据拉取；mock / live 自动切换。
- **SaaS 对接配置界面**（设置页）：接口地址 / API Key 配置（密文存储）、连接状态、测试连接、主体 ↔ SaaS 门店编码映射、「立即同步」并写入财务流水（自动去重）。
- **财务报表导出**：资产负债表、利润表、现金流量表、科目余额表、税负汇总，支持单店与集团合并双维度，导出 Excel / CSV / 打印 PDF。
- **国标税务申报表**：增值税及附加税费申报表（小规模 & 一般纳税人）、企业所得税月（季）度预缴 A 类表（小微优惠自动套用）、财务报送表；支持按主体 + 年度 + 季度生成、导出与打印。
- 新增数据表：`saas_config`、`saas_entity_map`、`compliance_nodes`。
- 新增 API Key 加密工具 `lib/crypto.ts`。
- 侧边栏新增「税务申报」入口。
