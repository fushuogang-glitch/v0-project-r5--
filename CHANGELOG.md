# 更新日志

本项目所有重要变更均记录于此文件。版本号遵循语义化版本（SemVer）：主版本.次版本.修订号。

## [0.2.0] - 2026-05-31

### 新增
- 登录页改版为左右双栏 SaaS 风格：左侧深色品牌展示区（运营数据指标），右侧登录表单。
- 登录页新增"记住我""密码显隐切换"等交互。
- 登录页底部新增「关联 NOTA CoreControl™ 全智能 SaaS 系统」说明。

### 优化
- 登录页右侧登录区调整为约 1/3 宽度，整体表单尺寸收紧，兼容电脑、平板、手机三端。
- 左侧面板品牌点缀颜色由紫色统一为主题红色系。
- 左侧标语更新为「美业连锁化智能财务管理系统」（财务软件定位）。
- 设置页密钥卡片标题由「财务 Agent 密钥」更名为「SaaS API 密钥」。

### 修复
- 修复 `app/actions/hr.ts` 在 `'use server'` 文件中非法导出常量导致 `/org` 页面 500 的问题（职级常量迁移至 `lib/hr-constants.ts`）。

## [0.1.0]

### 新增
- 美业连锁财务管理系统初始版本：组织架构、员工/工资管理、税务提醒、Agent API 等。
